# Changelog

All notable changes to PokeNav are documented here.

## v0.1.0 — Initial Release (2026)

### Features

- **Encyclopedia** — Game-first Pokemon encyclopedia with cross-linked species, moves, abilities, items, types, regions, and locations
- **Trainer System** — Trainer articles, battle archives, team breakdowns, and appearance-level browsing
- **Game-Aware Browsing** — Browse Pokemon, trainers, and locations scoped to a specific game version
- **National Dex** — Full national Pokedex with search, filtering, and stat comparison
- **Battle Simulator** — Type matchup analysis, team builder, and battle result cards
- **Data Pipeline** — Python scripts for generating encyclopedia JSON from PokeAPI and Bulbapedia data
- **Security Hardening** — Content Security Policy, import validation, and input sanitization
- **GitHub Pages** — Static deployment with SPA routing support
